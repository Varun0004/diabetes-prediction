# Diabetes Risk Prediction

This project builds a machine learning model to predict the risk of diabetes from patient health indicators such as glucose level, BMI, age, and blood pressure. The work is implemented in a Google Colab notebook so it can be easily run and experimented with.

## Dataset
- Public diabetes / health indicators dataset (binary label: diabetes or no diabetes).
- Includes features like glucose, BMI, age, blood pressure and related medical variables.

## What I did
- Performed data cleaning, basic exploratory data analysis, and feature scaling.
- Trained and compared multiple classifiers (e.g. Logistic Regression, Random Forest, KNN).
- Evaluated models using accuracy, precision, recall, F1‑score, and ROC‑AUC.
- Identified important features to understand which health indicators are most associated with diabetes risk.

## Tech stack
- Python
- Google Colab
- Pandas, NumPy
- Scikit‑learn
- Matplotlib / Seaborn

## How to run
1. Download the notebook or open it directly in Google Colab.
2. Upload the dataset (or connect to the same source used in the notebook).
3. Run all cells to reproduce preprocessing, model training, and evaluation.
